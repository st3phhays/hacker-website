Hack to fly in Minecraft survival mode

  1. Turn on cheats.
  2. Put it on education mode.
  3. Go into Minecraft and go into chat and do /ability mayfly true

Teleporting Hack

  1. Go into Minecraft
  2. Go into chat
  3. Type /tp @p 5~5~5 true

How to make a slide with a command block

  1. Go into Minecraft
  2. Go into chat
  3. Type /give @p command_block
  4. Place it down in Minecraft
  5. Right click on it
  6. Type /summon zombie
  7. Put it on Repeat, Always active.
  8. Put another command block
  9. Enter the command /kill @e [type=zombie]
  10. Put it on Repeat, Always active.
  11. Exit the command block
  12. You will see a slide of zombies

How to get super strength in Minecraft

  1. Go into Minecraft
  2. Go into chat
  3. Type /effect @p strength 9999 255 true
